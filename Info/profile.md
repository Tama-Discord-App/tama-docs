Use `/profile` to see your complete profile, which includes:

### Pet Information
- Current pet with its live icon state
- Pet name and type
- Hunger, energy, and love stats
- Time since stats were last updated

## Profile Buttons

Your profile has interactive buttons:

### Inventory
Opens your item inventory where you can:
- View all owned items
- See item quantities
- Use items on your pet
- Check item descriptions

### Badges
View your earned badges and achievements:
- Completed achievements show in color
- Locked achievements show as grayscale
- See requirements for locked achievements
- Track your progress

### Mailbox
Check messages and notifications:
- System messages
- Event notifications
- Important updates
- Clear old messages

## Achievements

Complete server-wide goals to earn rewards.

{% hint style="info" %}
Achievements are **server-wide** - your entire Discord server works together to complete them!
{% endhint %}

### Viewing Achievements
Use `/achievements` to see all available achievements:
- Progress toward completion
- Completion status (bronze, silver, gold tiers)
- Reward information (bells/stars for server members)
- Achievement categories

Achievements track collective server activity like:
- Total pets adopted in the server
- Total games played
- Server milestones

When your server completes an achievement, all active members may receive rewards!

{% hint style="info" %}
Members must have been in the server for at least a week to be eligible for achievement rewards!
{% endhint %}

### Earning Rewards
When your server completes an achievement:
- Server members may receive bells/stars
- Achievement tier increases (bronze ‚Üí silver ‚Üí gold)
- Progress toward higher-tier rewards

## Inventory Management

### Using Items
From your inventory:
1. Select an item
2. Click "Use"
3. Confirm the action
4. Item effect applies to your pet

### Item Organization
Items are grouped by type:
- Food items
- Energy items
- Toys
- Special items

You can easily see what you have available for pet care.
